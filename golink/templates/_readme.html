Golinks allow assigning a short name for a URL prefix:

  • go/example → {{ 'http://example.com/foo/'|urlize }}

Once created, anything placed after the name will be appended to the resulting URL used for the redirect:

  • go/example/bar → {{ 'http://example.com/foo/bar'|urlize }} (http://example.com/foo/ + bar)
  • go/example#test → {{ 'http://example.com/foo/#test'|urlize }} (http://example.com/foo/ + #test)
